\# 提示词构建器 (Tag Prompt Builder)



一款基于 PyQt6 的桌面工具，用于高效构建、组织并导出 AI 图像生成（如 Stable Diffusion / NovelAI）所需的提示词标签。支持层级标签库、勾选互斥、随机池抽取、排序面板、预设管理以及复制历史等特性。



\## 功能特性



\- \*\*层级标签库\*\*  

&#x20; 树状结构管理标签，支持文件夹分组、互斥选择（如发色、瞳色）以及标签的别名、描述和参考链接。



\- \*\*搜索与收藏\*\*  

&#x20; 实时过滤（中/英文）标签，快速定位；可收藏常用标签，独立面板管理。



\- \*\*动态排序与预览\*\*  

&#x20; 右侧分层排序面板自动同步已勾选标签，可拖拽调整顺序；实时预览中文提示词并检测括号配对。



\- \*\*随机抽取与互斥池\*\*  

&#x20; 创建随机池，配置抽取数量与重复策略；在排序中插入随机槽，生成时自动从池中抽取且遵守互斥规则。



\- \*\*预设保存与加载\*\*  

&#x20; 支持文件夹预设（保存选中状态和排序结构）和词组预设（快速勾选一组标签），一键复用历史组合。



\- \*\*历史记录\*\*  

&#x20; 自动记录每次复制的提示词，可从菜单恢复并重新勾选。



\- \*\*标签详情\*\*  

&#x20; 点击标签文本可查看中英文名、Wiki 链接、别称及简介。



\## 安装与运行



\### 环境要求

\- Python 3.10+

\- PyQt6

\- pyperclip（剪贴板操作）



\### 安装依赖

```bash

pip install -e .

```

或在项目根目录执行：

```bash

pip install -r requirements.txt

```



\### 启动程序

```bash

python -m tag\_prompt\_builder.main

```

或者直接运行：

```bash

python tag\_prompt\_builder/main.py

```



首次启动后，会在 `%APPDATA%/tag\_prompt\_builder` (Windows) 或 `\~/.local/share/tag\_prompt\_builder` (Linux) 下创建用户数据目录，并生成默认标签库。

\## 使用指南



1\. \*\*勾选标签\*\*  

&#x20;  在左侧标签树中勾选需要的标签，右侧排序面板将实时显示。



2\. \*\*调整顺序\*\*  

&#x20;  在分层排序面板中通过拖拽或上下移动按钮改变标签顺序，影响最终提示词输出。



3\. \*\*互斥选择\*\*  

&#x20;  勾选互斥文件夹（如发色）下的多个标签时，会弹出对话框询问替换或保留两者。



4\. \*\*随机槽\*\*  

&#x20;  先在菜单“文件”→“管理随机池”中创建池并添加标签。然后在排序面板右键“插入随机槽”，配置抽取参数。生成预览时随机槽会自动抽取结果。



5\. \*\*预览与复制\*\*  

&#x20;  右侧预览区显示中文组合，并自动检查括号匹配。点击“复制到剪贴板”即可导出英文标签串。



6\. \*\*预设管理\*\*  

&#x20;  - \*\*保存文件夹预设\*\*：保存当前所有勾选的标签和排序结构。  

&#x20;  - \*\*加载文件夹预设\*\*：一键恢复之前的状态。  

&#x20;  - \*\*词组预设\*\*：在文件夹右键菜单中保存/加载该文件夹下的勾选组合。



7\. \*\*导入提示词\*\*  

&#x20;  通过“文件”→“从提示词导入”，粘贴逗号分隔的英文标签，程序会自动匹配并勾选。



\## 配置与自定义



\- 修改 `resources/default\_tags.json` 可自定义初始标签库结构（需在首次启动前放置或手动替换用户数据目录下的 `tags.json`）。

\- `app\_config.py` 中可调整数据存储路径或预留语言/主题选项。

\- 随机抽取的行为（互斥逻辑）可在 `ui/widgets/layered\_sort\_panel.py` 的 `perform\_random\_slot` 中调整。
