 标签提示词构建器 (Tag Prompt Builder)

基于 PyQt6 的桌面工具，用于高效构建、组织并导出 AI 图像生成（如 Stable Diffusion / NovelAI）所需的提示词标签。支持层级标签库、互斥标签组、图形化排序面板、随机池抽取、方案保存以及剪贴板历史等特性。

---
 功能特性

- 层级标签库
  树状结构管理标签，支持多级标签组分类、互斥选择（如发色、瞳色），以及每个标签的中英文名、Wiki 链接和描述。

- 搜索与收藏
  实时中/英文过滤，快速定位标签；收藏夹独立面板，一键查看常用标签。

- 图形化排序与权重
  右侧图形排序面板自动同步已勾选标签，以可拖拽的树形节点展示；支持拖拽调整顺序，双击设置权重（输出格式 `(tag:1.2)`），实时预览提示词并检查括号配对。

- 随机抽取与互斥池
  创建随机池，配置抽取数量与重复策略；在排序中插入随机槽，生成预览时自动从池中抽取且遵守互斥规则。

- 方案与组合
  - 方案：保存当前所有勾选标签及其排序结构，一键恢复完整提示词布局。
  - 组合：仅保存一组标签 ID，快速勾选常用标签集。

- 剪贴板历史
  自动记录每次复制的提示词，可从菜单恢复并重新勾选。

- 标签详情与 Wiki
  点击标签可查看中英文名、别称、简介及 Danbooru Wiki 链接。

- 导入外部提示词
  支持粘贴逗号分隔的英文标签（包括 `(tag:1.2)` 等权重格式、括号嵌套），自动匹配并勾选；未匹配标签可一键追加到“未匹配标签”组。

---

 安装与运行

 环境要求
- Python 3.10 或更高版本
- PyQt6 ≥ 6.5
- pyperclip（剪贴板操作）

 安装
```bash
 克隆仓库
git clone https://github.com/yourusername/tag_prompt_builder.git
cd tag_prompt_builder

 安装依赖
pip install -e .
```

或直接安装 `requirements.txt`：
```bash
pip install -r requirements.txt
```

 启动
```bash
python -m tag_prompt_builder.main
```

首次启动时，程序会在用户数据目录（Windows: `%APPDATA%/tag_prompt_builder`，Linux: `~/.local/share/tag_prompt_builder`）自动创建默认标签库。

 使用指南

 1. 勾选标签
在左侧标签树中勾选所需标签，右侧图形排序面板会实时显示已选标签的树形结构。

 2. 调整排序与权重
- 拖拽节点调整顺序（标签组可整体移动）。
- 双击标签节点可设置权重（0.1 – 5.0），生成提示词时自动添加 `(tag:权重)` 格式。
- 通过右键菜单可插入随机槽，预览时自动从对应随机池抽取标签。

 3. 互斥标签组
标签组可设为互斥模式（如“发色”）。勾选新标签时弹出选择框：替换或同时保留。

 4. 预览与复制
- 下方预览区显示中文组合，并自动检查英文括号配对。
- 点击 “复制到剪贴板(英文)” 将英文标签串复制到剪贴板，同时自动记录历史。

 5. 方案管理
- 保存方案：菜单 → 文件 → 保存方案，将当前所有勾选及排序结构保存为方案文件。
- 加载方案：从列表中选择已存方案，一键恢复标签选择和排序布局。

 6. 组合管理
- 在标签组的右键菜单中选择 “保存为组合” 或 “加载组合”，快速保存/恢复该组下的勾选状态。

 7. 导入外部提示词
菜单 → 文件 → 从提示词导入，粘贴英文标签（支持权重、括号等格式）。程序自动匹配并勾选，未识别标签可一键添加至 “未匹配标签” 组。

 8. 剪贴板历史
点击菜单栏 “最近使用” 可查看、恢复已复制的提示词。

---

 配置与自定义

- 默认标签库：修改 `resources/default_tags.json` 可自定义初始结构（需在首次启动前替换，或手动覆盖用户数据目录下的 `tags.json`）。
- 用户数据路径：所有用户数据（`tags.json`、`recent.json`、`random_pools.json`、`presets/`）保存在 `app_config.py` 中定义的目录，可按需修改。
- 随机抽取逻辑：核心抽取算法位于 `ui/helpers/random_util.py`。
